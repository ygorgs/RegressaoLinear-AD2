A maioria dos homens saudveis tm nveis menores de 4ng/ml de sangue. A atividade para este checkpoint ser prever o nvel do PSA em pacientes atravs de uma regresso linear.

Os dados utilizados neste problema so referentes a exames em pacientes homens afim de identificar sintomas de cncer de prostata. O dataset composto por dez variveis, onde uma delas (psa) ser considerada nossa varivel resposta. Nosso objetivo final ser predizer a varivel psa a partir dos preditores disponveis.

Como campos do dataset, temos:
vol: volume do cncer
weight: peso do paciente
age: idade do paciente
bph: hiperplasia prosttica benigna
svi: invaso das vesculas seminais
cp: penetrao capsular
gleason: escore Gleason
pgg45: percentagem escore Gleason 4 ou 5
psa: antgeno especfico da prstata (esta a varivel resposta).

Obs.: A varivel train ser utilizada para diviso do dataset em treino e teste. Obs2.: Observe que foi aplicada a funo log em algumas variveis.

Treinando o modelo

A variveis de interesse que escolhi foram lcavol, svi e lcp. Para o treinamento do modelo, sero usados os dados de treino (dados_treino).

#definindo variveis
volume <- dados_treino$lcavol
invasao <- dados_treino$svi
penetracao <- dados_treino$lcp
psa <- dados_treino$lpsa

#obtendo a regresso linear
reg_linear <- lm(lpsa ~ lcavol+svi+lcp, data = dados_treino)

#sumrio do modelo
summary(reg_linear)

#predies
